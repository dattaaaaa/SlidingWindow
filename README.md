**Run / Preview**
- Open `index.html` directly in your browser (double-click or `File → Open`).
- From VS Code use the _Live Server_ extension to serve and auto-reload the page.
- Or run a simple HTTP server (PowerShell):

```powershell
python -m http.server 8000
# then open http://localhost:8000 in your browser
```
